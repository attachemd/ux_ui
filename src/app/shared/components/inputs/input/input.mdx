# Input

A fundamental UI component that allows users to enter and edit text. The `ft-input` is built with a "mixin-like" CSS architecture, offering high performance and deep customization through CSS variables.

## When to use
- To collect user data such as names, emails, or numbers.
- For search functionality.
- For sensitive data like passwords (using the `inputType="password"`).

## Design Tokens

The component uses several CSS custom properties for internal styling. You can override these for specific instances:

| Variable | Description | Default |
| :--- | :--- | :--- |
| `--_field-bg` | Background color of the input wrapper | `transparent` |
| `--_field-border-color` | Border color | `var(--ft-color-border-200)` |
| `--_field-height` | Height of the field | `var(--ft-unit-800)` |
| `--_field-font-size` | Font size of the text | `var(--ft-text-xs)` |
